## 🚀 Topics

| Topic | Path | Description |
|-------|------|-------------|
| Back-of-the-Envelope Math | `src/system-design/math/` | Numbers, latency, storage, QPS formulas |

## 🛠 Run Locally

```bash
npm install
npm start
```

## 🌐 Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import repo
3. Click **Deploy** — done!
